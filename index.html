目禅 - 专注工作，用眼有度
MuZen - Focus Mindfully, Rest Your Eyes

应用概述 / App Overview

目禅是一款专为现代工作者设计的用眼提醒应用，基于科学的20-20-20用眼法则，帮助长时间使用电子设备的用户养成健康的用眼习惯。

MuZen is an eye-care reminder app designed for modern professionals. Based on the 20-20-20 rule, it helps users who spend long hours on screens develop healthy eye habits.

核心功能 / Key Features

🔔 智能提醒系统 / Smart Reminders

  • 基于20-20-20法则：每20分钟休息20秒
  • 可自定义提醒间隔（5到60分钟）
  • 支持静音、声音、震动等提醒方式
  • 本地通知，无需联网
  • 20-20-20 rule: Rest for 20 seconds every 20 minutes
  • Customizable intervals (5–60 minutes)
  • Multiple alert options: silent, sound, vibration
  • Local notification, no internet needed

⏰ 灵活的时间管理 / Flexible Timer

  • 直观的倒计时显示
  • 优雅的进度环动画
  • 一键开始/暂停功能
  • Clear countdown interface
  • Elegant progress ring animation
  • One-tap start/pause

🎯 个性化设置 / Customization

  • 三种模式：标准(20分)、高效(25分)、轻松(15分)
  • 自定义休息时长（10–60秒）
  • 自动深色/浅色主题切换
  • 简洁直观的界面
  • Three modes: Standard (20min), Focus (25min), Light (15min)
  • Custom break duration (10–60 sec)
  • Auto light/dark theme
  • Minimal and intuitive UI

📊 使用统计 / Usage Stats

  • 每日休息次数统计
  • 简洁的数据展示
  • 帮助了解用眼习惯
  • Daily rest tracking
  • Clear data visualization
  • Helps build eye care habits
